=============
Installation
=============

to get the project do :

.. code-block:: bash

    git clone https://github.com/foxmask/django-th.git

or 

.. code-block:: bash

    pip install django-th



Dependencies
------------
* Python 3.4.0
* Django > 1.6
* django-th-rss >= 0.3.0
* django-th-evernote >= 0.3.0
* evernote SDK for python 3 https://github.com/evernote/evernote-sdk-python3
